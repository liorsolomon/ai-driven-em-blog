# AI-Driven EM Blog

Static site for *The AI-Driven Engineering Manager* — engineering leadership in the age of AI.
No build step: plain HTML/CSS, deployable as-is on Vercel.

## Structure
- `index.html` — homepage (hero, topics, latest articles, newsletter)
- `about.html` — about page
- `posts/*.html` — three starter articles
- `styles.css` — all styling
- `vercel.json` — clean-URL config

## Deploy
Connect this repo to Vercel (Framework Preset: **Other**, no build command, output = repo root)
and every push to `main` auto-deploys. See `DEPLOY.md`.
